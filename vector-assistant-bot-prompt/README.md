# Вектор Ассистент 🎯

Персональный ИИ-ассистент для предпринимателя: голосовые заметки, финансовый учёт, отчёты.

**Полностью бесплатно** — использует встроенные в браузер технологии (распознавание речи), собственный NLP-парсер русского языка и бесплатный хостинг.

## 📋 Шаг 1. Скачайте код приложения на компьютер

1. В этом проекте найдите кнопку **«Code»** (зелёная, вверху справа)
2. Нажмите **«Download ZIP»**
3. Распакуйте архив в любую папку на компьютере

---

## 📋 Шаг 2. Создайте новый репозиторий на GitHub

1. Откройте [github.com](https://github.com) и войдите в аккаунт
2. Нажмите **«+»** (вверху справа) → **«New repository»**
3. В поле **Repository name** напишите: `vektor-assistent`
4. Нажмите **«Create repository»**
5. На открывшейся странице нажмите **«uploading an existing file»**
6. Перетащите все файлы из распакованной папки (кроме папки `node_modules`)
7. Нажмите **«Commit changes»**

> ⚠️ Важно: не загружайте папку `node_modules` и файл `.env` — их быть не должно (они уже указаны в `.gitignore`)

---

## 📋 Шаг 3. Создайте бесплатную базу данных на Neon

1. Откройте [neon.tech](https://neon.tech) и нажмите **«Sign up»**
2. Войдите через GitHub (нажмите **«Continue with GitHub»**)
3. Подтвердите аккаунт (если попросит)
4. После входа нажмите **«Create project»**
5. В поле **Name** напишите: `vektor-assistent`
6. В поле **Postgres version** оставьте как есть
7. В поле **Region** выберите ближайший (например, `EU Central` или `Israel Central`)
8. Нажмите **«Create project»**
9. На открывшейся странице найдите блок **«Connection string»**
10. Нажмите кнопку **«Copy»** — скопируйте строку подключения

> Сохраните эту строку — она понадобится на следующем шаге. Она выглядит примерно так:
> ```
> postgresql://user:password@hostname:5432/dbname?sslmode=require
> ```

---

## 📋 Шаг 4. Подключите GitHub к Vercel

1. Откройте [vercel.com](https://vercel.com) и войдите в аккаунт
2. Нажмите **«Add New»** → **«Project»**
3. В списке репозиториев найдите `vektor-assistent` и нажмите **«Import»**
4. На странице настроек:
   - **Framework Preset**: Vercel определит автоматически (Next.js)
   - **Build Command**: оставьте как есть
   - **Install Command**: оставьте как есть
5. Прокрутите вниз до блока **«Environment Variables»**
6. В поле **Key** введите: `DATABASE_URL`
7. В поле **Value** вставьте строку подключения из Neon (из Шага 3)
8. Нажмите **«Deploy»**
9. Дождитесь завершения сборки (1–2 минуты)

> ✅ Если всё прошло успешно — вы увидите зелёную галочку и ссылку на ваш сайт.

---

## 📋 Шаг 5. Проверьте работу приложения

1. На Vercel нажмите на ссылку вашего сайта (выглядит как `https://vektor-assistent-xxxx.vercel.app`)
2. Откройте сайт в браузере **Chrome** или **Edge**
3. На главной странице нажмите **«Голос»**
4. Разрешите доступ к микрофону
5. Скажите: **«Завтра в 9 утра замер на Ленина 15»**
6. Нажмите **«Стоп»**, затем **«OK»**
7. Задача должна появиться в списке

> Если голос не работает — используйте текстовый ввод (напишите текст в поле и нажмите OK).

---

## 📋 Шаг 6. Создайте Telegram-бота

1. Откройте в Telegram [@BotFather](https://t.me/BotFather)
2. Нажмите **«Start»** (если ещё не запускали)
3. Напишите команду: `/newbot`
4. BotFather спросит имя — напишите: `Вектор Ассистент`
5. Затем спросит username — придумайте уникальный, например: `vektor_assistent_bot`
   (должен заканчиваться на `bot`)
6. BotFather пришлёт сообщение с токеном — **сохраните его** (пригодится позже)

---

## 📋 Шаг 7. Подключите веб-приложение к боту (Telegram Web App)

1. Вернитесь в [@BotFather](https://t.me/BotFather)
2. Напишите команду: `/newapp`
3. Выберите вашего бота (нажмите на его username)
4. Введите название для Web App (например: `Ассистент`)
5. Вставьте ссылку на ваш сайт с Vercel (обязательно с `https://`)
6. Готово!

---

## 📋 Шаг 8. Добавьте кнопку открытия Web App

1. В [@BotFather](https://t.me/BotFather) напишите: `/mybots`
2. Выберите вашего бота
3. Нажмите **«Bot Settings»** → **«Menu Button»**
4. Нажмите **«Configure menu button»**
5. Введите текст кнопки (например: `Открыть ассистента`)
6. Вставьте ссылку на ваш сайт с Vercel
7. Готово!

---

## 📋 Шаг 9. Пользуйтесь!

1. Откройте вашего бота в Telegram
2. Нажмите кнопку **«Открыть ассистента»** внизу экрана
3. Откроется ваше веб-приложение внутри Telegram
4. Нажмите **«Голос»** и диктуйте задачи или сделки
5. Бот всё распознает и сохранит

> 💡 **Совет:** голосовой ввод лучше работает при открытии приложения в браузере Chrome на телефоне или компьютере. Внутри Telegram голос может не работать из-за ограничений Telegram — в этом случае используйте текстовый ввод.

## 🎯 Как пользоваться приложением

### Голосовые задачи
Нажмите «Голос», скажите:
> «Завтра в 9 утра замер на Ленина 15, потом к 12 доставка окон на Партизанскую, в 16 звонок клиенту по вентиляции»

Бот создаст 3 задачи с правильными датами и временем.

### Финансовые сделки
На странице «Финансы» нажмите «Голос», скажите:
> «Продал кондиционер за 30 тысяч, купил за 17700, монтаж 5000, материалы 800»

Бот посчитает маржу и сохранит сделку.

### Несколько сделок сразу
> «Вчера продал два кондиционера, первый купил за 15, продал за 28, монтаж 4 тысячи, фреон и кронштейны 600. Второй купил за 22, продал за 35, монтаж 5500, материалы 900»

Бот создаст 2 сделки и посчитает итоговую маржу.

### Отчёты
На странице «Отчёты» выберите период (день/неделя/месяц) — увидите выручку, маржу, детализацию по категориям. Кнопка «Экспорт CSV» скачает файл для Excel.

## 🔧 Технологии (для интересующихся)

- **Next.js 16** — веб-фреймворк
- **PostgreSQL** — база данных (Neon, бесплатно)
- **Drizzle ORM** — работа с базой данных
- **Tailwind CSS** — стилизация
- **Web Speech API** — бесплатное распознавание речи (встроено в Chrome/Edge)
- **Собственный NLP-парсер** — извлечение дат, времени, сумм из русского текста (без LLM)

## ❓ Частые вопросы

**В: Голос не работает внутри Telegram?**
О: Откройте приложение в браузере Chrome (на телефоне или компьютере) — там голос работает. Внутри Telegram используйте текстовый ввод.

**В: Сколько это стоит?**
О: Полностью бесплатно:
- GitHub — бесплатно
- Vercel — бесплатно (хватает для личного использования)
- Neon — бесплатно (0.5 ГБ базы данных)
- Telegram — бесплатно

**В: Можно ли использовать на телефоне?**
О: Да, открывайте ссылку Vercel в Chrome на телефоне.

**В: Данные сохраняются?**
О: Да, все задачи и сделки сохраняются в базе данных PostgreSQL на Neon.

**В: Как обновить приложение?**
О: Внесите изменения в код на GitHub — Vercel автоматически пересоберёт и обновит сайт.

**В: Как добавить напоминания?**
О: В текущей версии напоминания не реализованы. Это можно добавить позже.
